Thanks for visiting!

ISS Pass Times is my final project from an Intro to Web Development course. The site showcases my knowledge of HTML, CSS, and JavaScript. If you're intereseted, you can view the code for the site on my github.

What inspired the website?

During the Web Development course our final project was to make a wesbite using the knowledge learned during the course, specifcally Javascript and interacting with an API. After some thinking I wondered if I could make a site that would tell me the best time to see the ISS traverse the sky from my backyard each night. After several evenings of work, I was able to find free resources that supplied the times the ISS would pass overhead. My ambitious goal originally had the site supplying basic weather info for clear skies, as well as if the pass times occur when the sky is dark. I quickly learned that an easy thought isn't as simple to code! I may add this functionality to the site someday, but do not have any current plans to do so.
